Taco Truck Database

The purpose of the project is to outline a database management system for La Cocina de la Abuela Taco Truck to implement for their company. There was a three-step process in creating the database management system – Conceptual Design, Logical Design, and Physical Design.

This is an Image of the Entity Relationship Diagram (ERD) used to conceptualize and map out what data needs to be modeled and how each object is related to one another.
